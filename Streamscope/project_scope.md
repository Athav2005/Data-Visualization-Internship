# Project Scope - StreamScope

## Objective
To analyze the Netflix Movies and TV Shows dataset to uncover trends in content growth, genre distribution, and country-level contributions.

## Milestone 1 Focus
- Dataset loading
- Data cleaning and preprocessing
- Handling missing values
- Removing duplicates
- Normalizing categorical features
- Creating derived features (year_added)

## Success Metrics
- No duplicate records
- No missing values
- Standardized categorical formatting
- Clean dataset exported for EDA

## Output
Cleaned dataset: netflix_cleaned.csv
